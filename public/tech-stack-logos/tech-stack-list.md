# Tech Stack List For Web Agency Services

## Frontend Development

### Core Technologies
- **html5** - HTML5 markup language
- **css** - CSS styling
- **js** - JavaScript programming language
- **typescript** - TypeScript for type-safe development

### JavaScript Frameworks & Libraries
- **reactjs** - React for building user interfaces
- **nextjs** - Next.js for server-side rendering and static sites
- **vuejs** - Vue.js framework
- **nuxtjs** - Nuxt.js for Vue applications
- **angular** - Angular framework
- **sveltejs** - Svelte framework

### CSS Frameworks & UI Libraries
- **tailwind** - Tailwind CSS utility-first framework
- **bootstrap** - Bootstrap framework
- **bootstrap5** - Bootstrap 5 latest version
- **material-ui** - Material-UI React components
- **sass** - Sass CSS preprocessor

### Mobile Development
- **reactjs** - React Native (using reactjs logo)
- **flutter** - Flutter for cross-platform mobile apps
- **ionic** - Ionic framework
- **android** - Android native development
- **swift** - Swift for iOS development

## Recommended Stack Combinations

### Modern Full-Stack (Recommended)
- Frontend: **reactjs**, **nextjs**, **typescript**, **tailwind**
- Backend: **nodejs**, **nextjs** (API routes)
- Database: **mongodb** or **postfresql**
- Cloud: **aws** or **vercel**
- Tools: **git**, **github**, **figma**, **postman**

### Mobile App Development
- Cross-platform: **flutter** or **reactjs** (React Native)
- Backend: **nodejs**, **firebase**
- Database: **firebase** or **mongodb**
- Cloud: **firebase** or **aws**

### E-Commerce/CMS Projects
- Frontend: **nextjs**, **reactjs**, **tailwind**
- Backend: **nodejs** or **php**, **laravel**
- Database: **mysql** or **postfresql**
- Cloud: **aws**
- Tools: **wordpress** (if needed)

### Enterprise Applications
- Frontend: **angular** or **reactjs**, **typescript**
- Backend: **java**, **python**, **django**
- Database: **postfresql**, **redis**
- Cloud: **aws**, **azure**, **google-cloud**
- DevOps: **docker**, **aws-ec2**
